# OSS 云存储

> 七牛云文件仿百度网盘文件夹管理，上传下载，删除

### 功能简介

1. 仿百度网盘（有悬浮窗）
2. 拖拽上传（现只能单个文件，或者多个文件上传，不能直接上传文件夹）
3. 下载进度
4. 托盘图标
5. 右键菜单
